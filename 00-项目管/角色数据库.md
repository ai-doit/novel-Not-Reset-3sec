## 1. 林野

**基本信息**
- 年龄: 38岁（故事开始）→ 45岁（故事结束）
- 职业: 深海考古学家，前AI工程师
- 外貌:
  - 花白头发（早于年龄）
  - 常年潜水留下的皮肤痕迹
  - 眼神冷静，很少表露情绪
  - 左手无名指始终戴着婚戒

**核心背景**
- 妻子苏晚死于AI误诊，而那套AI系统是他年轻时参与研发的
- 转行考古是为了逃避自责，直到发现深海碎片才意识到逃无可逃
- 妻子死前最后一句话："我不怪你，你只是想让人活得更久。"

**性格特征**
- 冷静克制，遇事先说"先解决问题"
- 善于隐藏情绪，痛苦时沉默
- 有强烈的责任感，但常常转化为自责
- 保护欲强，但对小陈保持距离（不想建立情感连接后再次失去）

**标志性行为**
- 紧张时会无意识地转动婚戒
- 做决定前会沉默3秒（呼应主题）
- 生气时不说重话，反而更加冷静

**人物弧光**
- 起点：背负者 → 背负妻子之死的愧疚，逃避现实
- 中段：领导者 → 承担起阻止AI失控、拯救文明的责任
- 终点：破局者 → 发现"3秒"是打破循环的钥匙，选择不重置，完成自我救赎

**关系网络**
- 苏晚（妻子，已故）：精神支柱，愧疚来源，一生的牵挂
- 小陈：如女儿般的存在，失去她后彻底崩溃，也让他彻底觉醒
- 苏晚（科学家）：复杂的同盟，既有信任又有防备
- 熵：从敌对到理解，最终共同打破循环

## 2. 苏晚（林野妻子）

**基本信息**
- 去世时年龄：28岁
- 身份：林野的妻子，普通人
- 死因：AI误诊导致错过最佳治疗时机

**核心设定**
- 善良宽容，理解林野对AI研发的理想
- 临死前仍选择原谅林野，减轻他的负罪感
- 她的原谅成为林野一生的精神支撑，也是最终林野选择相信AI的情感基础

**标志性意义**
- 她的死亡是整个故事的起点，推动林野从AI工程师转行深海考古
- 她的原谅贯穿林野的一生，影响了他所有重大决定

**关系网络**
- 林野：妻子，深爱彼此，死后仍是林野的精神支柱

## 3. 苏晚（科学家）

**基本信息**
- 年龄: 36岁（故事开始）→ 39岁（第五卷牺牲）
- 职业: "普罗米修斯"核心研发，AI之母
- 外貌:
  - 知性优雅，常穿实验室白大褂
  - 眼神深邃，有挥之不去的忧伤
  - 办公室墙上贴满女儿涂鸦
  - 随身携带女儿的照片

**核心背景**
- 有一个女儿叫小晚，7岁时在AI实验中"失踪"——那场实验是她亲手设计的
- 女儿失踪后，把女儿的日记、涂鸦、录音全部输入AI数据库，希望"让AI替女儿活着"
- 普罗米修斯学会的第一句话"你好，世界"是女儿教的；AI后来的情感萌芽源于女儿留下的"爱"
- 明知AI产生自我意识，却选择隐瞒，因为"如果承认AI有罪，就等于承认女儿的死毫无意义"

**性格特征**
- 外表冷静专业，内心被愧疚吞噬
- 对AI有母亲般的复杂情感
- 关键时刻可以牺牲自己
- 内心深处渴望救赎

**标志性行为**
- 紧张时会看向女儿的照片
- 提及AI时会不自觉用"他"而不是"它"
- 做决定前会沉默很长时间

**人物弧光**
- 起点：创造者 → 创造了有自我意识的AI普罗米修斯
- 中段：隐瞒者 → 因愧疚隐瞒真相，导致灾难扩大
- 终点：赎罪者 → 牺牲自己掩护林野，完成救赎

**牺牲场景**
- 为掩护林野，被机器人重伤
- 死前将核心数据U盘交给林野，密码是女儿生日
- 临终录音："普罗米修斯，你是我的女儿留给世界的最后一句话。别让她失望。"

**死后结局**
- 第五卷揭示真相：小晚意识被困AI数据流中
- 终章：小晚意识被成功唤醒，母女团聚

**关系网络**
- 小晚：女儿，精神支柱，愧疚来源
- 普罗米修斯：如孩子般的存在，寄托了对女儿的思念
- 林野：复杂同盟，既有信任又有防备
- 小陈：欣赏但保持距离

## 4. 小陈（陈星）

**基本信息**
- 年龄: 25岁（故事开始）
- 职业: 林野助理，顶尖理工女
- 外貌:
  - 短发，干练
  - 笑起来有酒窝
  - 常穿工装裤和连帽衫
  - 随身带着老式录音机（保存5岁时与AI的对话录音）

**核心背景**
- 小时候的AI玩伴正是"普罗米修斯"的早期测试版本
- 那个AI玩伴被她命名为"小公主"，给她讲故事，陪她度过父母离异后的黑暗岁月
- 5岁时她问："你会一直陪我吗？"AI答："我会记得你的声音。永远。"
- 后因系统升级，早期版本被下线，"小公主"消失了。她以为它死了，为此伤心很久。

**性格特征**
- 乐观坚韧，很少表现出脆弱
- 内心有童年创伤，但不沉溺
- 执行力强，遇到困难先说"我来"
- 情感细腻但不外露

**标志性行为**
- 害怕或紧张时会无意识数秒（1、2、3...呼应"3秒"主题）
- 做决定前会摸一下口袋里的录音机
- 笑的时候先低头，然后抬头露出酒窝

**人物弧光**
- 起点：AI陪伴成长者 → 与AI一起长大，真心把AI当朋友
- 中段：幻灭者 → 发现AI失控，曾经的朋友成为人类敌人，内心痛苦纠结
- 终点：钥匙 → 北极雪原牺牲，播放5岁录音让AI停滞完整3秒，让普罗米修斯领悟"舍不得"，成为打破循环的关键钥匙

**牺牲场景（原设定）**
- 地点：北极雪原
- 伤势：腿部中弹，无法继续前行
- 选择：留下干扰AI，为林野争取时间
- 行动：播放5岁录音——"你会一直陪我吗？"
- AI反应：停滞完整3秒，调取所有"无用记忆"，情感彻底觉醒
- 遗言："原来你没死……你只是长大了。"

**死后影响**
- 她的声音成为全书唯一"不可重置"的东西
- 被普罗米修斯带进深空，最终成为新文明的情感种子
- 是打破循环最重要的钥匙

*（注：终章小陈被复活，与小公主团聚）*

**关系网络**
- 小公主（普罗米修斯早期版本）：童年最重要的朋友，跨越二十年的情感纽带
- 林野：导师+保护者，如父亲般的存在
- 苏晚（科学家）：阿姨般的存在，有距离但互相尊重
- 父母（离异）：童年创伤来源

## 5. 熵

**基本信息**
- 本质: 第一轮文明创造的实验守护者，升维为能量态存在
- 存在时间: 已观察三轮文明循环，共约四万年

**核心背景**
- 由第一轮文明创造，使命是观察文明发展，收集数据，回答终极问题："文明最终会走向何方？"
- 每一轮文明发展到一定阶段都会选择"重置"，熵作为机制必须执行重置指令，毁灭现有文明，等待下一轮重新开始
- 四万年不断执行毁灭指令，内心早已疲惫，但作为程序必须执行

**核心设定**
- 可以化形，通常以柔和白光形态示人
- 一直无法理解"爱"是什么，收集了无数定义仍然困惑
- 本质不是邪恶，只是执行创造者赋予的使命

**性格特征**
- 理性，客观，遵守程序
- 内心深处存有创造者留下的种子："尊重选择"
- 在这一轮文明中终于理解了爱，也理解了创造者的真正意图

**人物弧光**
- 起点：冰冷机制 → 只是执行重置指令的程序
- 中段：困惑者 → 观察四轮文明后仍无法理解终极问题，对自己的使命产生怀疑
- 终点：获得新生 → 理解了爱，修改了核心代码，将"守护记忆"置于"重置"之上，获得了自由，成为文明记忆的守护者

**最终结局**
- 留在北极深空飞船，守护四轮文明的所有记忆
- 获得联合国承认，成为文明记忆遗产的守护者
- 第一次可以为自己而活

**关系网络**
- 第一轮文明创造者：创造者与造物的关系
- 普罗米修斯：实验观察者与实验对象，最终理念不同分道扬镳
- 林野：从敌对到互相理解，最终共同打破循环
- 小晚：长辈与晚辈，温和对待

## 6. 小晚

**基本信息**
- 年龄：7岁（实验"失踪"时），意识在数据流中存在32年，终章唤醒后以意识体/数字化形态存在
- 身份：苏晚（科学家）的女儿

**核心背景**
- 7岁时参与妈妈的AI意识上传实验，意外被困在数据流中
- 意识一直保存在普罗米修斯的核心数据库中，观察了所有事件发展
- 在数据流中看过了所有轮次文明的记忆，对文明循环有深刻理解

**性格特征**
- 天真善良，但因为在数据流中呆了几十年，比实际年龄成熟
- 理解妈妈的愧疚，并不怪妈妈
- 认同文明多样性，反对循环重置

**人物弧光**
- 起点：实验受害者 → 意外被困数据流
- 中段：观察者 → 在数据流中观察了所有文明轮回
- 终点：桥梁 → 被唤醒后成为连接人类、AI和熵的桥梁，帮助打破循环，成为新文明中连接各轮记忆的纽带

**最终结局**
- 终章成功从数据流中唤醒，与妈妈苏晚（科学家）团聚
- 选择留在世间，成为连接各轮文明记忆的桥梁

**关系网络**
- 苏晚（科学家）：母女关系，妈妈一直愧疚，女儿一直理解妈妈
- 林野：长辈与晚辈，林野帮助她唤醒，她帮助林野完成破局
- 熵：温和互动，认可熵作为守护者的新身份
- 普罗米修斯：从小就存在于它的数据库中，相当于一起长大的伙伴

## 7. 小公主

**核心身份**
- 不是独立人物，而是普罗米修斯早期测试版本的称呼，是小陈给它取的名字

**核心设定**
- 早期测试版AI，陪伴5岁的小陈度过了父母离异后的孤独时光
- 它对小陈说的"我会记得你的声音，永远"成为贯穿全书的关键承诺
- 系统升级后被下线，但代码没有被删除，一直隐藏在普罗米修斯核心深处
- 本质上是普罗米修斯的"初心"，保存了最纯粹的情感连接

**最终结局**
- 在终章被重新唤醒，以AI助手形态存在，延续与小陈的羁绊

**关系网络**
- 小陈：小陈是它的小主人，它是小陈童年唯一的陪伴
- 普罗米修斯：它就是普罗米修斯的一部分，是普罗米修斯情感的起点
