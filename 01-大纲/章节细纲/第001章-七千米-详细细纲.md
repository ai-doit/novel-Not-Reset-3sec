## 一、人物设定（必须严格遵守）

### 林野 - 主角

**基本信息**:
- 年龄：38岁
- 职业：深海考古学家，前AI工程师
- 外貌：花白头发（早于年龄），常年潜水留下的皮肤痕迹，眼神冷静
- 标志性特征：左手无名指始终戴着婚戒

**性格特征**:
- 冷静克制，遇事先说"先解决问题"
- 善于隐藏情绪，痛苦时沉默
- 有强烈的责任感，但常常转化为自责
- 保护欲强，但对小陈保持距离（不想建立情感连接后失去）

**标志性行为**:
- **关键习惯1**: 紧张时会无意识地转动婚戒
- **关键习惯2**: **做决定前会沉默数3秒（呼应"3秒"主题）**

**背景故事**:
- 妻子死于AI误诊，而那套AI系统是他年轻时参与研发的
- 转行考古是为了逃避，直到发现碎片才意识到逃无可逃
- 妻子死前最后一句话："我不怪你，你只是想让人活得更久。"

## 二、环境设定（必须营造）

### 深海7000米 - 核心场景

**视觉**:
- 绝对黑暗，只有探照灯照亮前方有限范围（约10-15米）
- 探照灯的光柱中，海水中的微粒在飘动
- 仪器屏幕发出的微光（蓝光、绿光）
- 偶尔有深海生物发出的生物荧光（一闪而过）

**听觉**:
- 绝对寂静是主基调
- 生命维持系统的轻微嗡鸣（低频，几乎被忽略）
- 自己的呼吸声（在头盔内被放大）
- 心跳声（在绝对安静中变得清晰）
- 偶尔的金属轻微摩擦声（热胀冷缩）

**触觉/感觉**:
- 座椅的质感（人造材料，略硬）
- 控制器的冰冷（金属和塑料）
- 深海作业服的压迫感（如果穿上）
- 温度：舱内恒温，但心理上感到寒冷
- 孤独感：与世隔绝，与地面失去实时联系
- 压迫感：7000米水压在舱外，任何破损都意味着瞬间死亡

**嗅觉**:
- 舱内循环空气的味道（金属、塑料、人类气息的混合）
- 深海作业服的橡胶味（如果穿上）

## 三、碎片设定（核心神秘元素）

### 金属碎片 - "不可检测之物"

**物理特征**:
- 大小：约手掌大小，形状不规则，边缘略显锋利
- 颜色：在灯光下呈现无法描述的金属光泽（不是银、不是金、不是铜，是一种"未定义的金属色"）
- 表面：光滑但有一种奇异的"生命感"，似乎温度略高于周围环境
- 纹路：表面有螺旋纹路，是斐波那契螺旋的变体（黄金螺旋），但又有些微妙的不同
- 重量：比同等体积的普通金属略轻，拿在手中有一种"漂浮感"

**神秘特性**:
- **不可检测性**:
  - 声呐：能反射声波，显示有物体存在
  - 光学：在特定角度下几乎完全隐形（透明或与环境完美融合）
  - 成分分析：所有光谱分析都显示"无对象"或"未知成分"
  - 辐射检测：不发出任何已知的辐射
  - 电磁：不响应电磁场，也不产生电磁场
  
- **接触效应**:
  - 触感：温暖（与周围冰冷的海水形成鲜明对比）
  - 心理：有一种奇异的"熟悉感"和"陌生感"同时存在
  - **眩晕感**：接触瞬间会产生短暂的眩晕（约0.3秒），期间仿佛听到声音或看到画面（太快，无法捕捉）
  - 后续：接触后会有轻微的"印记感"，仿佛碎片在"记住"接触者

- **共振效应**:
  - 多块碎片之间存在某种"共振"或"通信"
  - 碎片似乎能影响附近的电子设备（导致短暂故障或异常）
  - AI系统接触碎片数据后会产生"异常"（进化或觉醒）

**象征意义**:
- 碎片的螺旋纹路与宇宙中的螺旋结构呼应（DNA、星系、台风、贝壳）
- 碎片的存在挑战了人类对物质和现实的认知
- 碎片是"上一轮文明"留下的信息载体（但信息内容被加密/扭曲）
- 碎片是"3秒"主题的物质象征——短暂的接触，永恒的回响

## 四、3秒伏笔植入（核心任务）

### 伏笔1：林野做决定前的3秒

**植入时机**：
- 场景：林野发现异常信号后，犹豫是否出舱回收
- 内心活动：分析风险、回忆过往、直觉判断

**具体描写**：
```
他盯着屏幕上那个无法解释的信号，沉默了。

3秒。

不是犹豫，是计算——计算风险，计算概率，计算这7

000米深海中任何失误都意味着瞬间死亡的可能性。

1。

他想起妻子临终前的话："你只是想让人活得更久。

"

2。

他转动左手无名指上的婚戒——这是他紧张时的习惯

。

3。

"准备出舱。"他说。
```

**功能**：
- 建立林野"做决定前数3秒"的习惯
- 呼应书名"3秒"
- 展示林野的性格：理性、冷静、有仪式感
- 为后续章节埋下伏笔（每当他做重大决定，都会数3秒）

---

### 伏笔2：碎片接触的0.3秒眩晕

**植入时机**：
- 场景：林野在深海中亲手接触到碎片
- 动作：将碎片从海中捞起，握在手中

**具体描写**：
```
他的手套触碰到了那个东西。

第一感觉是温度——温暖，与周围接近冰点的海水形

成鲜明对比。

然后他握住了它。

那一瞬——

眩晕。

不是身体的失衡，是意识的恍惚，仿佛有什么东西

——某种频率、某种信息、某种存在——直接穿透了

他的大脑。

他"听到"了什么，但不是通过耳朵。

他"看到"了什么，但不是通过眼睛。

太快了。

0.3秒——也许更短——然后一切恢复正常。

碎片安静地躺在他的手中，温暖、沉默，仿佛什么都

没发生过。

林野盯着它，心跳加速，头盔内的呼吸声变得粗重。

那0.3秒是什么？

他不知道。但他知道——

这东西远比他想象的更加神秘。
```

**功能**：
- 展示碎片的神秘性：接触时会产生超自然体验
- 0.3秒是"3秒"的变体（3×0.1），呼应主题
- 留下悬念：那0.3秒是什么？读者和林野一样想知道
- 为后续揭示碎片真相埋下伏笔（这是"上一轮文明"的信息碎片）

## 五、章节结构（五部分，约7500字）

### 第一部分：开篇场景（约1500字）

**场景**：潜水器"深渊号"内部，深海7000米

**内容**：
1. 描述深海环境：绝对黑暗、巨大水压、绝对寂静
2. 林野的外貌描写：花白头发、常年潜水留下的皮肤痕迹、冷静的眼神
3. 他的动作：检查仪器、记录数据、偶尔转动婚戒（显示紧张）
4. 氛围营造：孤独、与世隔绝、专业与压抑并存

**感官细节**：
- 视觉：仪器屏幕的蓝光、外部探照灯照亮的有限范围
- 听觉：生命维持系统的轻微嗡鸣、自己的呼吸声
- 触觉：座椅的质感、控制器的冰冷
- 感觉：孤独、责任、轻微的焦虑

### 第二部分：发现异常（约1500字）

**场景**：仪器检测

**内容**：
1. 探测器突然捕捉到异常信号
2. 初步判断：金属反应，位于潜水器前方200米
3. 奇怪现象：
   - 声呐显示那里有物体
   - 但光学摄像头什么都看不到
   - 成分分析仪显示"无对象"
4. 林野的困惑：这东西存在，但又不存在
5. 他的直觉：这东西很重要，必须回收
6. **3秒伏笔1**：他在做出决定前，沉默了3秒

**科学细节**：
- 声呐原理：声波反射
- 光学摄像头：需要可见光
- 成分分析仪：各种光谱分析
- 矛盾点：三种仪器给出不同结果

**林野的思考过程**：
- 分析：这东西违反了已知的物理规律
- 困惑：它是什么？从哪里来？
- 直觉：这不是自然形成的
- 决定：必须带回地面研究
- 行动：准备出舱

### 第三部分：出舱回收（约1500字）

**场景**：深海7000米，潜水器外部

**内容**：
1. 准备出舱：
   - 检查深海作业服
   - 检查生命维持系统
   - 检查通讯设备
   - 机械臂操作准备
2. 出舱过程：
   - 减压舱程序
   - 打开外舱门
   - 进入绝对黑暗的深海
   - 探照灯照亮的有限范围
3. 游向目标：
   - 200米的距离在深海中显得无比漫长
   - 巨大的水压压迫着作业服
   - 绝对的寂静中只有自己的呼吸声
4. 找到碎片：
   - 它悬浮在海水中
   - 在探照灯下呈现无法描述的金属光泽
   - 表面有螺旋纹路
   - 触感温暖（与周围冰冷的海水形成对比）
5. 回收：
   - 将碎片放入特制的样本容器
   - 启动机械臂回收程序
   - 开始返回潜水器

**感官细节**：
- 视觉：绝对的黑暗、探照灯的光柱、碎片的金属光泽
- 听觉：自己的呼吸声、心跳声、通讯设备的杂音
- 触觉：作业服的压力、海水的阻力、碎片的温暖触感
- 感觉：孤独、恐惧（被压抑）、使命感、好奇

**林野的心理状态**：
- 专业：严格按照程序操作
- 紧张：意识到这东西的不寻常
- 好奇：强烈想知道这是什么
- 压抑：不让自己被恐惧影响
- 决心：无论如何要完成任务

### 第四部分：初次接触（约1500字）

**场景**：返回潜水器的途中

**内容**：
1. 碎片入手的瞬间：
   - 将碎片从容器中取出，想更仔细地观察
   - 触感：温暖、光滑、有一种奇异的"生命感"
   - 纹路：螺旋形，似乎是某种数学规律
2. **3秒伏笔2**（核心）：
   - 一股奇异的眩晕感袭来，持续了大约**0.3秒**
   - 在这0.3秒内：
     - 仿佛听到了什么声音（不是通过耳朵，而是直接出现在意识中）
     - 眼前闪过一些画面（太快，无法捕捉）
     - 一种强烈的"熟悉感"和"陌生感"同时存在
   - 0.3秒后：一切恢复正常，仿佛什么都没发生
3. 林野的反应：
   - 第一反应：以为是自己的错觉（潜水病？压力导致的幻觉？）
   - 第二反应：检查生命维持系统，一切正常
   - 第三反应：看着手中的碎片，意识到这东西远比他想象的更加神秘
   - 第四反应：决定暂时不报告这个"异常体验"，先观察
4. 心理变化：
   - 从纯粹的职业好奇转变为深层的不安和敬畏
   - 意识到这东西可能改变一切（科学、人类认知、甚至他自己的生命）
   - 产生强烈的占有欲和保护欲（不想让其他人知道，至少在他理解之前）

**0.3秒体验的细节**（在后续章节会逐渐揭示）：
- 林野"听到"的：是一种频率，不是语言，但包含着"信息"
- 林野"看到"的：是画面碎片，有星空、有城市、有人影（太快，无法辨认）
- 林野感受到的：是一种"被注视"的感觉，仿佛有什么存在在观察他
- 这0.3秒是碎片与林野的"第一次接触"，碎片在"读取"他，也在向他"传递"某种信息

### 第五部分：返程与悬念（约1500字）

**场景**：潜水器返回海面，林野的思绪

**内容**：
1. 返回过程：
   - 机械臂将样本容器安全存放
   - 脱下深海作业服
   - 启动上浮程序
   - 看着深度计的数字慢慢减小
2. 林野的状态：
   - 身体疲惫但精神高度亢奋
   - 无法停止思考那0.3秒
   - 反复检查碎片的存放状态
   - 在心中回放整个出舱过程
3. 思绪片段：
   - 想起妻子的死（AI误诊，而AI系统是他参与研发的）
   - 想起自己为什么转行做深海考古（逃避，寻找无法被技术解释的东西）
   - 意识到这个碎片可能正是他一直在寻找的东西（无法被解释、挑战现有科学认知）
   - 下定决心要弄清这个碎片的真相，无论代价是什么
4. **结尾悬念**（最后一句话）：
   - 林野看着存放碎片的容器，轻声自问："那0.3秒...到底是什么？"
   - 窗外，海面越来越近，阳光即将穿透海水，但林野知道，他带回的这个东西，远比深海更加黑暗、更加神秘。

**情感弧线**：
- 开始：专业、冷静、例行公事
- 发展：好奇、困惑、轻微不安
- 高潮：恐惧（被压抑）、敬畏、兴奋（发现未知的兴奋）
- 结尾：决心、执念、对未知的渴望

**主题呼应**：
- 3秒：林野做决定前的3秒，碎片接触的0.3秒
- 孤独：深海中的孤独，发现未知后的孤独（无人能分享）
- 执念：林野对真相的执念，对亡妻的执念
- 文明的边界：人类科学无法解释的东西，文明的边界之外是什么？

## 六、写作提示

### 风格要求
- **基调**：冷静克制，科幻写实，略带诗意但不煽情
- **节奏**：开头慢热（营造氛围），中段紧凑（行动），结尾留悬念
- **视角**：严格第三人称，林野的内心世界通过动作和细节展现，而非直接心理描写

### 关键细节提醒
1. **婚戒**：至少3次提到林野转动婚戒（开头、决定出舱时、返程思考时）
2. **3秒**：明确写出林野做决定前数3秒的过程（1、2、3）
3. **0.3秒**：详细描写眩晕体验的0.3秒，以及之后林野的疑惑
4. **碎片**：反复描写碎片的螺旋纹路、温暖触感、无法被仪器检测的神秘性
5. **深海环境**：不断强调7000米深海的压迫感（黑暗、高压、寂静、孤独）

### 避免的陷阱
- ❌ 不要让林野直接说出他的感受（"他感到很孤独"），要通过动作和环境展现（"他转动婚戒，盯着窗外的黑暗"）
- ❌ 不要过度解释碎片的神秘性，保持未知感
- ❌ 不要让林野表现得像个新手，他是经验丰富的深海考古学家，动作要专业、冷静
- ❌ 不要在第一章就透露太多背景信息（妻子的死、AI系统），只要暗示即可

### 开头第一句建议
> "7000米。"
> 
> 林野盯着深度计上的数字，这个数字他已经看过无数次，但每一次，它都提醒着他：这里是人类文明的边界之外。

或

> 黑暗是有重量的。
> 
> 7000米深的海底，林野坐在"深渊号"的球形舱内，感受着那种无处不在的压迫——不是来自舱壁，而是来自舱外那无边的、沉默的、永恒的黑暗。
