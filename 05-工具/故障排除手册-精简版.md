# 故障排除手册

> OpenClaw多Agent写作系统常见问题及解决方案

---

## 快速解决

### Agent无法启动
```bash
openclaw agent config validate main-writer
openclaw service restart
```

### 任务卡住
```bash
openclaw agent stop main-writer --force
openclaw agent restart main-writer
openclaw task retry <task-id>
```

### GitHub同步失败
```bash
git remote -v
git push origin main
# 如果认证失败，检查SSH密钥或Token
```
