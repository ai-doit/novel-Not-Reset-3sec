# 小说导出格式指南

> 将Obsidian中的Markdown小说导出为PDF、Word等出版格式

---

## 📦 导出工具选择

| 工具 | 输出格式 | 优点 | 缺点 | 适用场景 |
|------|---------|------|------|---------|
| **Obsidian PDF Export** | PDF | 保留Obsidian样式、支持目录 | 需要安装插件 | 最终出版 |
| **Pandoc** | PDF/Word/EPUB | 专业级排版、支持LaTeX | 需要命令行操作 | 正式出版 |
| **Obsidian Word Export** | Word | 保留格式、易于后期编辑 | 插件可能不稳定 | 提交编辑 |
| **Markdown to PDF** | PDF | 简单快速 | 样式简单 | 快速预览 |

---

## 🛠️ 方案一：Obsidian插件导出（推荐）

### 安装 PDF Export 插件

1. 打开 Obsidian
2. 设置 → 社区插件 → 浏览
3. 搜索 "PDF Export"
4. 安装并启用

### 导出单卷

1. 打开要导出的卷（如 `02-正文/第一卷-深海来信`）
2. 在文件列表中，按住 `Cmd/Ctrl` 选择所有章节文件
3. 右键 → 导出为 PDF
4. 选择保存位置，命名如 `第一卷-深海来信.pdf`

## 🛠️ 方案二：Pandoc导出（专业级）

### 安装Pandoc

```bash
# macOS
brew install pandoc

# 安装LaTeX（用于PDF生成）
brew install --cask mactex
```

### 导出脚本

创建 `05-工具/导出脚本/导出全本.sh`:

```bash
#!/bin/bash

PROJECT_DIR="/Users/derek/Documents/novel-不可重置的3秒"
OUTPUT_DIR="$PROJECT_DIR/output"

mkdir -p "$OUTPUT_DIR"

# 合并所有章节
cat $(find "$PROJECT_DIR/02-正文" -name "*.md" | sort) > "$OUTPUT_DIR/全本-合并.md"

# 导出PDF
pandoc "$OUTPUT_DIR/全本-合并.md" \
    -o "$OUTPUT_DIR/不可重置的3秒-全本.pdf" \
    --pdf-engine=xelatex \
    -V CJKmainfont="Noto Serif SC" \
    -V geometry:margin=2.5cm \
    -V linestretch=1.8 \
    --toc \
    --toc-depth=2

# 导出Word
pandoc "$OUTPUT_DIR/全本-合并.md" \
    -o "$OUTPUT_DIR/不可重置的3秒-全本.docx" \
    --toc \
    --toc-depth=2

echo "✅ 导出完成！"
echo "   PDF: $OUTPUT_DIR/不可重置的3秒-全本.pdf"
echo "   Word: $OUTPUT_DIR/不可重置的3秒-全本.docx"
```

赋予执行权限并运行:
```bash
chmod +x "$PROJECT_DIR/05-工具/导出脚本/导出全本.sh"
bash "$PROJECT_DIR/05-工具/导出脚本/导出全本.sh"
```

## 📋 导出检查清单

导出前，确保:

- [ ] 所有章节已最终定稿（v3-定稿或final）
- [ ] 所有链接（[[...]]）已正确解析
- [ ] 图片和附件已正确引用
- [ ] 目录层级正确（# ## ###）
- [ ] 分页符已适当添加（使用 `\newpage` 或 `---`）

导出后，检查:

- [ ] PDF/Word文件可以正常打开
- [ ] 目录链接可点击跳转
- [ ] 字体显示正常（特别是中文）
- [ ] 页眉页脚正确
- [ ] 页码连续
