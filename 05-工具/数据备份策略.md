# 数据备份策略

> 多层次备份方案确保数据安全

---

## 备份层次

| 层次 | 方式 | 频率 | 保留时间 |
|------|------|------|----------|
| 本地Git | 自动提交 | 每30分钟 | 永久 |
| GitHub | 远程同步 | 实时 | 永久 |
| Time Machine | 系统备份 | 每小时 | 30天 |
| 外部硬盘 | 冷备份 | 每周 | 长期 |

## 自动备份脚本

```bash
#!/bin/bash
# 保存为: backup.sh

cd /Users/derek/Documents/novel-不可重置的3秒

# Git自动提交
git add .
git commit -m "auto: $(date '+%Y-%m-%d %H:%M')"
git push origin main

# 创建本地备份副本
cp -r . ../novel-backup-$(date +%Y%m%d)

echo "✅ 备份完成: $(date)"
```

## 恢复流程

```bash
# 从Git恢复
git checkout HEAD -- file.md

# 从备份恢复
cp ../novel-backup-20260321/file.md .

# 从Time Machine恢复
# 打开Time Machine，选择文件，点击恢复
```
